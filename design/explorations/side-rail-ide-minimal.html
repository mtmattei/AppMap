A single path: Login → AppShell → Dashboard → PatientList → PatientDetail → MedAdministration. The final hop is declared but never observed this session — a high-risk medication flow with no runtime coverage.

Why this reads as VS, minimally: low-contrast hairlines instead of cards, a properties grid instead of stacked labels, collapsible <details> sections like Solution Explorer, sans labels paired with mono values, and exactly one accent (amber) — on the active-tab underline, the live pill, and the primary command. Everything else is graphite and negative space. It ports to XAML 1:1 because it already uses the Atlas palette; the only new pieces are the disclosure Expanders and the two-column grid.
